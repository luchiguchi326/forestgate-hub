# Forestgate Hub

Central repository for **Forestgate Entertainment** — projects, client notes, creative work, research, and AI conversation history.

## Structure

| Folder | Purpose |
|--------|---------|
| `clients/` | One subfolder per client — notes, deliverables, meeting history |
| `internal-tools/` | Scripts and automations used internally |
| `creative/` | Video, music, design, and media project tracking |
| `research/` | Research docs, reports, and reference materials |
| `conversation-history/` | Claude session logs and meeting notes |
| `_templates/` | Copy-paste starters for new clients, meetings, and sessions |

## Quick start

- New client? Copy `_templates/new-client/` into `clients/CLIENT-NAME/`
- - New meeting? Copy `_templates/meeting-note.md` into `conversation-history/meeting-notes/`
  - - New Claude session? Copy `_templates/claude-session.md` into `conversation-history/claude-sessions/`
