# Clients

One subfolder per client. Use `_templates/new-client/` as a starting point when onboarding a new client.

## How to add a new client

Create a new folder: `clients/CLIENT-NAME/` and add a `README.md`, `meeting-notes/`, and `deliverables/` subfolder.
