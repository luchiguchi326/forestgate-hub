# Internal Tools

Scripts, automations, and utilities used by Forestgate internally.

## Subfolders

- `automations/` — Scheduled or triggered scripts
- `snippets/` — Useful code snippets and one-liners
- `templates/` — Reusable document and email templates

## Adding a new tool

Create a subfolder with a short name and a `README.md` explaining what it does and how to run it.
